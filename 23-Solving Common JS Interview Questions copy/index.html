Consider the following JavaScript code


  class Vehicle {
    constructor(name, type) {
      this.name = name;
      this.type = type;
    }
  
    getName() {
      return this.name;
    }
  
    getType() {
      return this.type;
    }
  }
  
  class Car extends Vehicle {
    constructor(name) {
      super(name, 'car');
    }
  
    getName() {
      return 'It is a car: ' + super.getName();
    }
  }
  
  let myCar = new Car('Tesla');

What does the extends keyword do in this context?

It is used to create a class that is a child of another class


What is the purpose of the super keyword and how is it used here?

It is used to call the constructor of the parent class


What will myCar.getName() return and why?

It is a car: Tesla


How would you add a method to the Car class that overrides the getType method from the Vehicle class? 

getType() {
    return 'car';
  }
